Wiring the screen

" target="supplies">The screen is designed to be plugged on top of the Raspberry Pi using the GPIO pins, but we're going to build an extension that goes from the Raspberry Pi on the base to the screen inside the box above, using 26 " target="supplies">male to female dupont cables.

  • One by one, plug in cables from the Raspberry Pi to the screen, passing through the hole of the assembly.
  • You need to wire the thirteen leftmost pins in both rows, 26 pins in total.
  • Our goal is to reproduce exactly the connections if the screen were to be plugged on top of the Raspberry Pi.
Don't forget to pass all the cables through the assembly like this.
